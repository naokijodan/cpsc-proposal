7月から新品のおもちゃや服をアメリカで売るには — 新しいルールをわかりやすく

2026年7月からアメリカで変わるルール（CPSC新規制）を調べて、どうしたらいいか考えてみました

公開日: 2026年4月8日作成: naokijodan

1. どういうこと？ — ルールが変わる話

2026年7月8日から、日本からアメリカにモノを送るときのルールが変わります。

アメリカには「CPSC（シーピーエスシー）」という、危険な商品から消費者を守るための役所があります。この役所が「アメリカに商品を送る時は、前もって『これは安全ですよ』という書類を出してね」と決めました。

対象になるのは、主に次の3つです。

  • 子ども向けの商品（12歳以下が使うもの。おもちゃ、子ども服など） → CPC という書類が必要
  • 大人向けの服（Tシャツ、ジーンズなど） → GCC という書類が必要
  • 危険な可能性のある一般商品（自転車、ヘルメット、ボタン電池を使う商品など） → GCC という書類が必要
CPC と GCC の違い: CPCは厳しい書類で、アメリカが認めた検査所で実際に検査した結果が必要です。GCCはゆるい書類で、自分で「安全ですよ」と宣言すればOKなケースが多いです。
⚠書類がない荷物は、アメリカの税関で止められたり、返送されたり、最悪はeBayのアカウントが止められてしまうかもしれません。安い商品でも同じです。

2. 最初に考えたこと

新品のシルバニアファミリーやトミカなどを、アメリカに売りたいとします。でも、これって「子ども向けのおもちゃ」ですよね。ということは、上のルールだと CPC という書類が必要になってしまいます。

そこで最初に、こう考えました。

  • 「これは大人のコレクターグッズとして売ります」と言えば、子ども向けじゃないから規制を逃れられるのでは？
  • → でも、それはちょっとズルい気がします。誰が見てもシルバニアは子どものおもちゃです。
  • じゃあ、書類（CPC）を自分で作れないかな？

これを詳しく調べてみました。

以前の提案書（2026年3月29日版）では、eBayの出品ページの「対象年齢」を変えるなどの対応策を検討していました。今回の資料は、そのあとに「そもそも書類をどうするか」を調べた考察です。

3. 調べてわかったこと

3-1. おもちゃ・フィギュアのこと

「書類を自分で作る」は無理でした。

  • 正確に言うと、書類の紙だけは自分で作れます。でも、その書類に書く「安全です」という根拠の検査結果が、アメリカが認めた特別な検査所でちゃんと検査したものじゃないとダメなんです。
  • メーカーのホームページに「安全」と書いてあっても、それは証拠になりません。本物の検査報告書が必要です。
  • おもちゃは検査項目がたくさんあります。鉛が入っていないか、有害な化学物質が入っていないか、小さな部品が取れて飲み込まないか、燃えやすくないか…など。
  • 検査費用は、1つの商品につき 7万円〜22万円くらい。色や材料が多いともっと高くなります。
  • シルバニアファミリーやトミカは、どう見ても子ども向けのおもちゃなので、「大人のコレクター向けです」と言い張るのはムリがあります。
⚠商品が明らかに子ども向けなのに「17歳以上向け」と嘘のラベルを貼るのは、アメリカから見れば「ルール逃れ」とみなされて、もっと大きなトラブルになる可能性があります。

3-2. 服のこと

大人用の服は、わりとゆるいです。

  • 大人用の服は、自分で「これは安全ですよ」と宣言するだけでOKな場合が多いです（これが GCC という書類）。
  • 特に、ポリエステル・ウール・ナイロン・アクリルなどで作られた服は、検査自体が免除されます。
  • 厚手の綿（ジーンズ、厚手のシャツなど）も免除になります。
  • ただし、薄手の綿（薄いTシャツ、浴衣など）や薄手の絹（スカーフ、着物など）は注意が必要です。

子ども用の服は厳しいです。

  • 子ども用（12歳以下向け）の服は、おもちゃと同じくCPC書類が必要で、検査も必要です。
  • 特に、子ども用のパジャマや寝巻きは一番厳しいです。燃えにくさの検査があって、普通の綿100%のゆるっとした服はほぼNGになります。

中古の服は？

  • アメリカのルールでは、中古品の再販は検査も書類も免除されています（CPSCの公式ページにそう書いてあります）。
  • ただし、2026年7月からの新しい申告手続きで、中古品をどう申告するかの具体的な方法は、まだはっきりしていません。
⚠中古品についての注意: アメリカの公式ルールでは、中古品は検査や書類が免除されています。でも、eBayから日本のセラーに届いた2026年7月向けのお知らせには、中古品を免除する、とは書かれていません。eBayが独自に「中古でも書類を出してね」と求めてくる可能性もあるので、実際にどうなるかは今後の公式発表待ちです。

5. ではどうする？ — 4つの選び方

今のところ考えられる方法は4つあります。どれか1つに絞るのではなく、組み合わせて使うのが現実的です。

  1. 中古品だけを扱う アメリカのルールでは中古品は検査や書類が免除されているので、いちばんシンプルな方法です。ただし、eBayが2026年7月以降に「中古でも書類を出してね」と言ってくる可能性もあるので、公式発表を見ながら慎重に進める必要があります。
  2. 新品を「箱なし中古」として出す 新品を開封して箱を捨てて「Used（中古）」として売る方法です。やっている人はいますが、アメリカ側の判断しだいで「これは中古とは言えないよ」と言われるかもしれないグレーゾーンです。
  3. 新品は大人向けコレクター商品として売る 「14歳以上対象」とはっきり書いて売る方法です。ただし、誰が見ても子ども向けの商品にはこの方法は使えません。高価な限定品や、飾り専用の商品など、本当に子どもが遊ばないものだけにしましょう。
  4. メーカーから直接仕入れるルートをつくる シルバニアファミリーを作っているエポック社など、メーカーから直接仕入れて、検査データも一緒にもらう方法です。時間と手間がかかりますが、本気で新品のおもちゃを売りたいならこれが正攻法です。
補足: アメリカ向けの販売をやめて、ヨーロッパ・イギリス・オーストラリアに販売先を変えるという選択肢もあります。2026年7月からこういう厳しい制度を始めるのは、今のところアメリカだけです。
